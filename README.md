##  Legends of the Terminal (Mega Project 2) (March 9th - 14th 2026)

A terminal RPG written in Python.

#  About the Project

This is my second “mega project” while learning Python.
I wanted to build something bigger than the usual beginner programs like calculators or number guessing games.

So I decided to make a terminal-based RPG where you can explore, fight enemies, collect treasure, sell items, and upgrade your character.

The idea was to simulate a small RPG world completely in the terminal using Python.

#  Features
    	•	Exploration system
    	•	Random enemy encounters
    	•	Boss fights
    	•	Inventory system
    	•	Treasure and item rarity
    	•	Shop and selling system
    	•	Character upgrades
    	•	Health and combat mechanics

#  How to Play

Run the game with python main.py.

Then use the menu options to:
	•	Explore the world
	•	Fight enemies
	•	Find treasure
	•	Sell items for Aether
	•	Upgrade your stats
	•	Fight bosses
	•	Manage your inventory

The goal is to grow stronger and survive the world.
